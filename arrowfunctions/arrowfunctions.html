Arrow Functions

The function counts the letters in a word and returns a string with the outcome.

Named Function

            function numLetters(word){
            return(`${word} has ${word.length} letters`);
        }

Function Expression

            const numLetters = function(word){
                return(`${word} has ${word.length} letters`);
            }

Arrow Function

            const numLetters = (word) => `${word} has ${word.length} letters`;
